## Requirements
You need to install [NodeJS](https://nodejs.org/) and [MongoDB](https://www.mongodb.com)

Then, run in 'web-server' folder:
``` console
npm install
```

## Run
``` console
npm start
```

## Test
``` console
npm test
```
